# Manage Landing Page

A responsive landing page built as part of a Frontend Mentor challenge. This project showcases a modern UI layout with structured sections, clean design and responsive navigation.

## Overview

The **Manage Landing Page** is a static frontend project designed to present a product in a clear and engaging way. It includes key sections such as a hero banner, features, testimonials and a call-to-action.

The app is fully responsive and deployed on **Vercel**.

## Features

- Responsive navigation with mobile toggle
- Hero section with call-to-action
- Feature highlights with structured layout
- Testimonials section with user feedback
- Call-to-action banner
- Footer with social media links and email form

## Tech Stack

- HTML5
- CSS3 (custom styling, responsive design)
- Flexbox / Grid

## Responsive Design

The layout is designed to adapt to different screen sizes:

- Desktop view with full navigation
- Mobile view with hamburger menu toggle
